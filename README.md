# Data Cleaning & Reporting Automation 📊✨

A sleek, beginner-friendly web application that automatically cleans messy datasets and generates insightful reports with just a click.

## Features
- **Modern Dashboard UI:** Built with a clean, premium interface featuring glassmorphism and smooth animations.
- **Data Preview:** Upload your CSV file (or use the built-in sample data) and immediately preview all rows in a scrollable table.
- **One-Click Data Cleaning:**
  - Automatically removes duplicate rows.
  - Fills missing numeric values with the column average.
  - Removes extra spaces from text fields.
  - Standardizes column names (lowercase and underscores).
- **Automated Reporting:** Generates summary statistics for all numeric columns.
- **Interactive Visualizations:** Renders beautiful Bar and Doughnut charts using Chart.js based on your dataset.
- **Downloadable Outputs:** Easily download your cleaned `.csv` data and a summary report `.txt` file.

## Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript, Chart.js
- **Backend:** Python, Flask
- **Data Processing:** Pandas

## How to Run Locally

1. **Clone the repository** (or download the files):
   ```bash
   git clone https://github.com/vinit201105-tech/data-cleaning-automation.git
   cd data-cleaning-automation
   ```

2. **Install the required dependencies:**
   Make sure you have Python installed, then run:
   ```bash
   pip install -r requirements.txt
   ```

3. **Start the application:**
   ```bash
   python app.py
   ```

4. **Open the project:**
   Click the link below to view the live application:
   **[https://data-cleaning-automation-cams.onrender.com](https://data-cleaning-automation-cams.onrender.com)**
